The FinanPlans (short for The Financial Plans,) is a free website to help you budget your next vacation. Using simple questions such as your destination, how long you will stay there, which quality meals you would like to have, etc. We will give you the amount of US Dollars you would need for this trip. We have used HTML, CSS, and JavaScript to create this website. If given more time, we would have added more destinations so that this website could be used all around the world for people to budget their trips.
